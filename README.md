# Legyen Ön is milliomos — egyszerű webes játék

Egyszerű, helyi HTML/CSS/JS játék, amely a "Legyen Ön is milliomos" stílusát utánozza.

Fájlok:
- index.html: a játék felülete
- css/styles.css: stílusok
- js/script.js: játéklogika és kérdések

Futtatás:
1. Nyisd meg a `index.html` fájlt a böngésződben (dupla katt vagy `Open File`).
2. Kattints a "Játék indítása" gombra.

Kiterjesztések:
- Több kérdés, időzítő, hangok vagy további jokerek könnyen hozzáadhatók a `js/script.js` fájlban.
